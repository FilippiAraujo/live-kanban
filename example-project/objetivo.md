# Objetivo do Projeto

Criar uma ferramenta de visualização Kanban local que funcione em localhost e permita gerenciar tarefas de qualquer projeto através de arquivos simples.

## Definição de Pronto

- ✅ Interface web simples e intuitiva
- ✅ Leitura de 4 arquivos padrão (objetivo.md, status.md, tasks.json, llm-guide.md)
- ✅ Drag-and-drop funcional entre colunas
- ✅ Persistência automática das mudanças
- ✅ Visualização do guia LLM para referência
- ⏳ Sistema rodando em localhost:3000

## Por Que Este Projeto?

Esta ferramenta resolve o problema de gerenciar tarefas de forma visual e permitir que IAs auxiliem no gerenciamento através de um guia estruturado.
